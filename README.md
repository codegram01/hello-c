# Hello C

Simple example code in C for learning.

## Feature

- Organize sources code
- Simple build and run
- Test, unit test

## Command

All magic in Makefile, read it

### Build: 

```sh
make 
```

### Run: 

```sh
make run
```

### Unit Test: 

```sh
make [path file test]
```

Ex: 

```sh
make person/person_test
```

### Clean

Clean Build 

```sh
make clean
```

### Beauty code

```sh
make lint
```
